只鹳狸药堂

  
人间之里。
平日里热闹的傍晚的村落，却鲜见有谁出来活动。
灵梦［不对劲，明明是晚餐时间了，连个人影都没见着。
  

```
     天气也很舒适，天已经五月了啊。］
```

  
灵梦［等一下！］
  
  
空无一人的街道上，取代往日来来往往的人与妖怪们的，是飘飘荡荡的灵们。
所见到的除了人类灵，也有虚无缥缈的妖怪的灵。
灵体们似乎没有自我，四处晃荡着。
  
  
魔理沙［啊啊，飘呀飘的。
  

```
       我说啊，这样子应该已经是异变了吧？］
```

  
灵梦［他们不是完整的灵。
  

```
     不如说是有人夺走了他们的意识。
     真是麻烦……］
```

  
灵们四散开来，漫无目的地游走。
灵梦［再这样都要贴到脸上来了。
  

```
     早些把引起这些事的主犯教训一顿吧。］
```

  
魔理沙［你今天倒很勤快啊。
  

```
       不过看起来也不算是很严峻的异变。］
```

  
灵梦［早些处理完早点回来休息啦。］
