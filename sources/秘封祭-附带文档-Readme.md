###### 游戏的游玩方法


<table><tbody><tr class="tt-content" id="=-15" data-pos="&#91;&quot;=&quot;,15&#93;"><td class="tt-ja" lang="ja"><div class="poem"><br>■７、ゲームの遊び方<br><br>　　「秘封祭」は、メッセージを読み進めることによって進行し、選択肢を<br>　選ぶことでそれぞれのルートに突入する、マルチエンドタイプのノベルゲ<br>　ームです。メッセージは、左クリック・スペースキー・エンターキーを押<br>　して進めてください。<br><br>　　繰り返しプレイが前提となり、プレイヤーである貴方は、様々な秘封倶<br>　楽部の結末と向かい合うことになります。<br><br>　　１ルートにおけるエンディング数は１～３程度で、概ね一本道のシナリ<br>　オとなります。ルートによって長さ・雰囲気・作成担当者などが異なって<br>　きます。<br><br>　　一定数のエンディングを達成することで、ルートが解放され、冒頭に新<br>　しい選択肢が追加されます。一度見た文章は紫色で表示されます。ぜひ全<br>　てのエンディングを集めてみてください。<br><br>　　（＊どうしても攻略に行き詰った方は、「ネタバレおまけ」フォルダに<br>　収録されている「エンディングリスト.txt」を参照してください）<br><br>　　ゲームを終了する場合は、右上の×ボタンを押すか、タイトル画面から<br>　「夢を見終える」を選択して終了してください。<br></div></td><td class="tt-zh" lang="zh"><div class="poem"></div></td></tr></tbody></table>
