# 模板:Image_label

格式模板

## 用法
```
{{Image label|x= |y= |scale= |text= |color=}}
```

  
x表示字符中心到左边距离，y表示字符中心到上边距离，x和y的单位为像素 (未找到链接)，scale为放大率（可省略不填），text表示文字，color表示文字色彩（可省略不填）。
