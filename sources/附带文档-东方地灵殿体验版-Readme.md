<table><tbody><tr class="tt-content" id="=-15" data-pos="&#91;&quot;=&quot;,15&#93;"><td class="tt-ja" lang="ja"><div class="poem">■７．交信強度<br><br>　左下のアンテナのようなゲージは、地上に残っている妖怪との交信強度です。<br>　地上に近い画面上に移動すると瞬時にマックスになり、それ以外では徐々に低下します。<br>　また、敵弾にかすっても（何故か）上昇します。<br><br>　交信強度の割合で得点アイテムの得点が決まります。<br><br>　その交信強度がマックスの時は、アイテムが自分に寄ってきて得点アイテムも<br>　最高得点になります。<br><br>　また、交信強度の最大値は敵弾にかすっていると僅かに上昇します</div></td><td class="tt-zh" lang="zh"><div class="poem">■７．信号强度<br><br>　左下角的信号槽是，与留在地上的妖怪的信号强度。<br>　向离地面近的画面上端移动时会瞬间变满，其余时刻会慢慢变弱。<br>　此外，擦弹（为什么呢）也可以使信号上升。<br><br>　信号强度的大小会影响蓝点的得分最大值。<br><br>　在信号强度为满的时候，所有道具会被自动收集，这时蓝点也是<br>　最大得分。<br><br>　此外，最大得分的倍数会随着擦弹而上升。</div></td></tr></tbody></table>
