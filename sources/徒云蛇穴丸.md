徒云蛇穴丸（同人角色） - 二次同人角色
### 角色介绍
  
 **徒云蛇穴丸** 是东方Project的致敬作《 **[连缘Project](./连缘Project.md)** 》系列中登场的角色，初次在《连缘蛇丛剑 (未找到链接)》中作为EX面Boss登场。
  
  
与[徒云八尾吕智](./徒云八尾吕智.md)一同诞生的，由徒云氏长老尸体化为的七名大蛇妖怪&#91;八尾吕智设定&#93; (未找到链接)之一。
  

- 八尾吕智是长老被斩下的『腕』化成的妖怪，而蛇穴丸是『瞳』化成的妖怪。[^连缘蛇丛剑设定]

  
性格单纯、老实、一根筋，因此容易被欺负和欺骗，但是 **对于所爱之人会展现出极大的热情** 。
  

- 火锅中最喜欢滑鸡锅。[^连缘灵烈传设定]

#### 生活状况
  
平日里默默守护着[徒云八尾吕智](./徒云八尾吕智.md)，尽管连互相之间血脉相连一事都还没说明，蛇穴丸自己就已经相当满足。[^连缘灵烈传设定]
  

- 有些时候还会做出躲在草丛里偷窥八尾吕智[^连缘灵烈传设定]的行为。

  
但是由于在《连缘灵烈传 (未找到链接)》中，跟踪八尾吕智时 **被[鹀黑巫鸟](./鹀黑巫鸟.md)发现** ，[&#91;灵烈传剧情&#93;](./连缘灵烈传～Reactivate_majestical_imperial-设定与剧情-故事背景.md)为了不让黑巫鸟向八尾吕智告发只好成为祂的奴隶为祂卖命。被鹀家三人（[鹀黑巫鸟](./鹀黑巫鸟.md)、[鹀颊赤](./鹀颊赤.md)、[鹀蒿雀](./鹀蒿雀.md)）使唤着。
  

#### 外貌特征
  
徒云蛇穴丸作为[徒云八尾吕智](./徒云八尾吕智.md)的血亲，在外貌上与八尾吕智有一些相似。但是也有一些不同点：比如八尾吕智是黑发黑瞳，而蛇穴丸是绿发红瞳。
  
  
形象比较接近于 **日本武士** 。单尾、手持一把 **剑和薙刀** 。
  
  
虽然蛇穴丸的种族同样是大蛇妖怪，但因为祂无论是年龄还是形象都比徒云八尾吕智“小”很多，于是在粉丝中有了“小蛇”的外号。

#### 人际关系
  
徒云蛇穴丸是与[徒云八尾吕智](./徒云八尾吕智.md)是同时诞生的妖怪之一。  

徒云家一共有七人，但是经历了悠久的岁月，在历史的洪流中生还下来的，只有八尾吕智和蛇穴丸两人。[^连缘蛇丛剑设定]
  
  
对于八尾吕智怀有单向的爱恋，但身为栋梁的八尾吕智却并不知道自己尚有血亲在世，心里始终注视着剑。[^连缘蛇丛剑设定]
  

- 正是因为[蛇丛剑](./蛇丛剑.md)斩断了蛇穴丸与八尾吕智的因缘，才导致祂们永远无法相认。[来源不明]


  
即使是永远不能相认，蛇穴丸依旧 **无条件** 深爱着八尾吕智， **愿意为祂做任何事** 。因为害怕与八尾吕智相认会扰乱对方生活，选择在不远处观望守护，甚至不敢在八尾吕智面前露面，始终没有说出口：二人其实是血浓于水的亲人，不过本人已对此相当满足。[^连缘灵烈传设定]
  

```
 也许，我生生世世都再也不能言明自己是祂的血亲了。
 但只要能一直守护着那位大人，就足够了。
```

  
在《连缘灵烈传 (未找到链接)》中被[鹀黑巫鸟](./鹀黑巫鸟.md)胁迫帮祂去收集灵魂，成为了祂的奴隶。
