<table><tbody><tr align="left" valign="top"><td style="min-width:200px;"><b><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">特典访谈<br>谈谈那些被本书冷遇的角色吧</div></div></b>
<div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">那个，有一些角色既没有在幻想乡缘起也没有在天狗的新闻中出现……尤其是绵月姐妹。很多人似乎都不知道她们是什么样的角色。我也还有一些其他想要更多情报的角色，所以我来帮您收拾屋子，请您透露一下。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">呃—。绵月姐妹是月之公主，丰姬的能力是「连接山与海的能力」，这是可以移动到任何地方的能力。重要的是她能够来往于月之都和地面之间，此处山指幻想乡，海指月上。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">啊，这个含义原来很具体。我原本对她只有吃桃子和拥有超级武器的印象……</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">我对秋★枝小姐也没有进行关于武器的说明，所以武器这方面是超级笼统（笑）。然后，妹妹依姬则是普通的角色。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">依姬给我一种加强版主人公角色的感觉。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">毕竟她是作弊角色。那并非科学的力量，而单纯只是她的能力。能请来的神明越强，她就能变得越强。她太强了，不是漫画也没法让她出场。因为游戏里不论是什么样的BOSS前提都是可以被击败。这算是游戏独有的对于角色的限制。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">原来如此，不过正因为依姬的登场，灵梦的部分能力才得以首次被具体地描写。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">从这个角度上来说，我觉得那个漫画最大的看点其实是三段式火箭相当于住吉三神的地方。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">月之都与幻想乡的接触今后不会再被刻画了吗？</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">很难说。要想让她们再出来，可能要等到人类再登一次月。我说的不是NASA<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">1</a></sup>，而是民间公司，一旦登月能够盈利，感觉人类立刻就能重返月球。过去登月是冷战推动的，现在大概是钱吧。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">虽然宇宙的梗有很多，不过进入月之都的手段，和去茨华仙的家也有些相似。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">不论是月之都还是仙人的家，其实都有些相似。住在月之都的人们，比起人类，更接近于仙人或者天人。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">这么一说，记得月之都上也有神明，而这些神明是月之都最位高权重的人吧？</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">在我心中月之都的设定是高贵的神明们所在的地方。相反，幻想乡则有着平易近人、或者说是土著一般的神明。我觉得神明中或许也有各种各样的派阀。而永琳正因为曾是月之都的一员，来到幻想乡之后她也不和其他人类进行接触。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">这下很多事情都说得通了。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">因此，其实绵月姐妹对于幻想乡一侧来说是不可触碰的。这是必要的措施。不过这两个人的性格很好玩，有很多创作的空间，所以倒是还想再创作一些关于她们的故事。毕竟幻想乡中的姐妹，妹妹基本上都坏掉了。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">ZUN先生没有妹妹，所以这是某种怨恨吗（苦笑）。</div></div>
<b><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">［绯想天角色］</div></div></b>
<div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">关于天人和龙宫使者还想向您请教一些。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">单纯只是因为和神奈子、圣、神子没什么交集才遭到了冷遇，天子的性格还是很有趣的。<br>谁能想到到了游戏最后会被所有人欺负。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">感觉她的角色刻画和琪露诺是一个级别的。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">这种脑子不太灵光的感觉的角色多了很多。这样的角色（在格斗游戏里）登场起来更方便，毕竟畏首畏尾的角色不太好让他们登场。换在（射击）游戏里，倒是只要强行让灵梦路过顺手击败就可以了。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">衣玖又如何呢？</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">关于这个解读周遭的能力，感觉完全没被活用，或者说正因为解读了周遭，才会受到幻想乡那些不看气氛的家伙们影响，让她变得更加奇怪。不过认真地说，这个解读周遭其实是能在空中游来游去的含义。因此，对话多少有点不搭调也没问题。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">原来如此。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">说起来衣玖的种族并没有使用片假名<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">2</a></sup>，而是用汉字写作「<span lang="ja">竜宮の使い</span>（龙宫使者）」，而龙宫则是存在的。那就是指月之都。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">所谓天人住在月之都是指这个意思啊。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">在设定上这些都是有不少联系的。</div></div>
<b><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">［中BOSS们］</div></div></b>
<div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">琪斯美和椛等等中BOSS呢？</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">琪斯美等等……不知道平时在干些什么。不过，她的形象是很凶暴的。二话不说就来砍人的头，然后把头放在桶里拿回去。那个头说不定也不是琪斯美的头。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">太吓人了！</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">那个脸确实是她本人的，我只是打个比方，说她这个角色有多讨厌。椛的话……白狼天狗有很多伙伴，而他们还拥有着千里眼，所以是非常方便的角色。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">然后明明书中收录了花果子念报，却找不到果的条目……</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">毕竟这是个只有文才知道的角色……文又不会把果的事情写成新闻。但是果有着文所不具备的能力。念写是接近于千里眼的能力，但是可以影像化。想到关键词立刻就能浮现相关的事情，如同Google一般的能力。关于果我虽然真的是一直都忘了（还没在幻想乡缘起和新闻中提及过的事情），但由于这本书的设定，我也完全无法让她登场，难以进行说明。</div></div>
<b><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">［华仙］</div></div></b>
<div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">——</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">幻想乡缘起里也没有茨华仙。</div></div><div class="tt-char tt-type-dialogue" lang="zh"><div class="poem">ZUN</div></div><div class="tt-zh tt-type-dialogue" lang="zh"><div class="poem">她也是个用起来十分方便的角色，但她的故事仍处于现在进行时，所以不太好在这里多言。因此，请大家多多支持漫画。</div></div></td><td width="200px"><div class="thumb infobox noclear" style="width:200px; margin:0 3px 0 1em;">
