控制 - 核心 - 爆发 - 先手 - 耐久
  
  
装备的智力属性不会给琪露诺带来任何加成，但装备中提供魔法回复和最大魔法值的属性仍可提供正常的加成。

由于特殊的机制，蓝量上限在没有最大魔法值物品的加持下只有108点，很容易关键时刻被[](./文件-THD2衰人卡.png.md)抽掉蓝量丧失战斗力，所以御币很好解决了这个问题，在中路对线的情况下可以选择更多[](./文件-THD2练习手套.png.md)的增加属性方便对拼和补刀
  


### 前期装备
```
[](./文件-THD2魔法使扫把.png.md) [](./文件-THD2很值钱的礼物.png.md) [](./文件-THD2闪耀的宝玉.png.md) [](./文件-THD2博丽御币.png.md)
```

  
  

[](./文件-THD2很值钱的礼物.png.md)这件装备因为属性很契合琪露诺可以出一个关键时刻用掉补出关键装备，[](./文件-THD2闪耀的宝玉.png.md)是必出的，保证蓝量的同时后续合成其他装备用，[](./文件-THD2博丽御币.png.md)提升打架的容错率
  
  
  

  

- 如果感觉蓝量不够用就出以上推荐装备，出加智力的装备没有效果，这个人物的智力恒定是9（有兴趣可以试一下，出多少智力装备，显示那里的结果永远等于9）


### 核心装备
```
[](./文件-THD2格斗扫把.png.md) [](./文件-THD2奇迹之星.png.md)
```

  
  

由于最大蓝量限制很容易被[](./文件-THD2光龙雷云钻.png.md)限制成超级兵，如果发现对方近战物理核心装备有[](./文件-THD2河童冲锋手枪.png.md)时需尽早买出[](./文件-THD2奇迹之星.png.md)或者让辅助出关键时刻可以救人或自救
  

- 注意2技能[](./文件-THD2琪露诺_02.png.md)在冰面释放才有效果，在没有冰面的平地上释放会眩晕自己，1技能[](./文件-THD2琪露诺_01.png.md)会在原地创造一个冰面，三技能[](./文件-THD2琪露诺_03.png.md)会朝前直线形成冰面，经过测试在对面移动时使用3技能[](./文件-THD2琪露诺_03.png.md)接2技能[](./文件-THD2琪露诺_02.png.md)（不要按得太快，否则会导致冰面还没出来就先摔倒了）能够使2技能[](./文件-THD2琪露诺_02.png.md)打出3段伤害（是的二技能显示伤害是每段造成伤害而不是总共造成伤害），所以可以在对线时，利用对手走位补刀的时候进行耗血，效果会很好，对线近战或者觉得能杀人可以加1级1技能[](./文件-THD2琪露诺_01.png.md)，对拼和追人都非常好用，如果线上压力比较大可以23对线，依靠23技能连招远距离补刀

- 大招是本游戏为数不多的魔免（[](./文件-THD2琪露诺_04.png.md)状态下可以免疫绝大部分的魔法伤害和控制，星熊勇仪触发3技能[](./文件-THD2星熊勇仪_03.png.md)不会击飞，但仍然造成格外伤害，击飞变为眩晕一段时间）


### 可选装备

琪露诺的身板比较硬，且技能AOE不俗，大招还附带THD2为数不多的技能免疫（类似BKB，无法防御[](./文件-THD2星熊勇仪.png.md)的大招释放及秒杀效果，在大招期间不会被[](./文件-THD2星熊勇仪_03.png.md)击飞，但是仍然受到额外伤害，击飞效果替换为眩晕时间）效果，主要玩法是排抗伤害团战搅屎棍，针对对方输出类型做防御装备，能够在团战中生存更长的时间，干扰对方后排。由于大招附带技能免疫效果，也提供另一种思路，在技能免疫期间击杀对方后排，可以出一些输出装备和切入装备快速击杀对方关键人物，如保命装备较多可以出[](./文件-THD2绯想剑.png.md)针对，回复高可以出[](./文件-THD2死神之大镰刀.png.md)针对，如果需要针对的少女能打能抗（一般这种少女为力量少女智力属性低下）可以做出[](./文件-THD2万宝槌.png.md)针对，配合25级攻速天赋和[](./文件-THD2月兔粉碎巨刃.png.md)很容易打出无限眩晕效果，如果大后期打不开局面可以尝试出[](./文件-THD2天丛云剑.png.md) ，由于大招的存在可以更多发挥这件装备的作用

优先级：R→E→W→Q，需要注意的是W只能在QE创造的冰面使用才会有效，否则会原地眩晕一段时间，一般使用E接W远程消耗，也可以拿来补刀清线，对拼时在远处可以先放E减速目标，靠近目标或者目标快要走出冰面范围时释放W眩晕，接近目标后释放Q技能减速，Q技能附带减攻速和移速，对拼绝大多数时候都占据优势，由于E技能距离比较远，即使线上对拼劣势也可以用技能补刀，注意大招的持续时间内是不能释放QWE的，由于琪露诺的技能多附带减速等效果，技能释放可以一次放完打满伤害，也可以分段释放增加DEBUFF的持续时间。
