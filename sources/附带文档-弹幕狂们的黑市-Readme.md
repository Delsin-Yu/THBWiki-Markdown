###### 问候


<table><tbody><tr class="tt-content" id="=-3" data-pos="&#91;&quot;=&quot;,3&#93;"><td class="tt-ja" lang="ja"><div class="poem">■１．挨拶<br>　　この度は、「バレットフィリア達の闇市場　～ 100th Black Market.」を<br>　手にとって頂き、まことに有難う御座います。<br><br>　　何処かピントのずれた世界に何処かゆるいキャラ達が、とてつもなく激し<br>　い弾幕を繰り広げるシューティングゲームになっています。<br><br>　　今作は繰り返し遊ぶことで、少しずつ楽になっていって、ストーリーも進む<br>　感じを目指したゲームです。<br>　　ムキになって続けていたら、気が付いたら弾幕ＳＴＧが上手くなっている<br>　と思いますよ。</div></td><td class="tt-zh" lang="zh"><div class="poem">■１．问候<br>　　非常感谢各位能入手<br>　「弹幕狂们的黑市　～ 100th Black Market.」。<br><br>　　这是一部在某个有些虚幻的世界中，一群比较悠闲的角色<br>　展开无比激烈的弹幕的射击游戏。<br><br>　　本作制作时的方向是通过反复游玩，来逐步降低<br>　难度，并推进故事的游戏。<br>　　如果突然上头开始不断挑战的话，回过神来想必<br>　能发现自己对弹幕STG更加擅长了。</div></td></tr></tbody></table>
