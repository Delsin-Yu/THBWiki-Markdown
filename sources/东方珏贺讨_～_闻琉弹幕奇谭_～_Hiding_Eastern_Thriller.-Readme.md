## Readme

  
  

[CoSP01]东方珏贺讨 ~ 闻琉弹幕奇谭 ~ Hiding Eastern Thriller  

  

大家好，我是TNW。  

简单讲一下常见问题以及游戏的操作方法。  

  

  

1.简介  

  

东方珏贺讨是由LuaSTG的群友合作完成的STG作品。可能存在画风不一致等问题，请见谅。  

  

  

  

  

2.常见问题Q&amp;A  

  

Q:为什么无法运行？  

A:游戏不支持winxp及以下的操作系统。  

&#160;&#160;无法运行或提示初始化失败可能是找不到有效的音频设备。  

&#160;&#160;如果提示无法加载文件可能是分辨率的问题，也可能是内存不足。  

&#160;&#160;除此之外如果依然无法运行请将engines.log文件发送给开发者。  

  

Q:为什么全屏报错或闪退？  

A:请调节分辨率。  

  

Q:为什么全屏会拉伸窗口？  

A:可能是显卡的问题，调节显卡设置。如果这样也不行可能没有特别好的方法。  

  

Q:为什么游戏会弹出带有红色叉叉的窗口，无论点哪个按钮游戏都会自动关掉？我能做些什么？  

A:因为游戏出现了错误，所以不得不终止运行。你可以把报错弹窗截图和engines.log文件发送给开发者。  

  

Q:为什么游戏会闪退？我可以做些什么？  

A:因为游戏出现了错误，但是错误没有被捕获，所以不得不强制终止运行。你可以把出错的时机告诉给开发者，最好给开发者提供大量的有用的信息，比如游戏的log.txt文件。  

  

Q:怎么开启Ex关卡？  

A:不续关通关Mainline即可。  

  

Q:游戏好难怎么办？  

A:游戏难度略高，不过可以在stage practice里面体验全部关卡。还请多多练习。  

  

  

  

若遇到任何问题，请加群 202476353 向我们反馈。  

  

  

  

  

3.操作方法  

  

方向键       向对应的方向移动  

Slow(Shift)  按住可以进入低速移动模式  

Shoot(Z)     确认/射击  

Spell(X)     取消/Spell攻击  

Special(C)   释放对应自机的Special技能  

Escape       暂停或返回游戏 在暂停界面按R可以推把  

Home         截图  

Control      播放replay时加速  

Shift        播放replay时减速  

  

以上的部分键位可以在启动器里修改
