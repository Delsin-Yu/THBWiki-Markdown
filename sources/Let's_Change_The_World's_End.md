## 评论
```
『東方ラウドは終わった』の声に否を突きつけるために――

全12サークルが結集し作り上げる、東方ラウドの最前線。
東方×ラウドロックアレンジコンピレーション
Let's Change The World's End
―アルバム概要―
 
このアルバムは
Draw the EmotionalとForeground Eclipseに影響を受けた主催者が

東方×Metalcore/Post-hardcore/Screamo/etc...

すべての東方ラウドファンへ捧げるため

『東方ラウドは死んだ？』
『否！　否！　断じて否！』

『このジャンルをこれから盛り上げるのは』
『終わりの結末を変えるのは俺達だ！』

『俺が――』
『俺達が東方ラウドだ！！』

という東方ラウドに対して熱い想いを（多分）持つ
同志サークルを募り、製作を決定した
東方ラウドアレンジコンピレーションアルバムである。
 
―主催者コメント―

東方×ラウドミュージックという衝撃を初めて僕に教えてくれたのは
Draw the EmotionalとForeground Eclipseでした。

様々なサークルが浮かんでは消えていく中で
『東方ラウドはもう終わった』という声を聞きました。

けれど
彼等が僕達の心に灯してくれた火は
今も確かに燃えている。
彼等が描いた景色にはきっと
まだまだ続きがあるように思えてならなかったんです。

だから
今もこんなにたくさんのサークルが活動している中で
東方ラウドは終わったなんて言葉、到底納得できないし。
今までこのジャンルを作り上げてきた彼等の代わりにはなれないけれど
僕達なりの景色は描いていけると思いました。
僕達が一人ひとりで叫んだだけじゃ届かないかもしれない。
ならいっそ
コンピという形で全員で叫んでみようと思い至りました。
```
