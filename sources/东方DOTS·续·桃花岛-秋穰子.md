```
 [](./文件-THD2风见幽香.png.md)：属于是线上特别强势的组合，秋穰子的远距离先手能让风见幽香更容易打出技能，并且都是具有强回复能力的人物可以用来打强消耗的对线，最经典的风见幽香一技能[](./文件-THD2风见幽香_01.png.md)配合[](./文件-THD2秋穰子_02.png.md)可以在白天回复恐怖的血量，大大的增加推进阵容的效率，是风见幽香比较好配合的辅助
```

```
 [](./文件-THD2铃仙·优昙华院·因幡.png.md)：比较好配合的远程敏捷核，除了能保线上更好发育外，[](./文件-THD2秋穰子_04.png.md)配合[](./文件-THD2铃仙·优昙华院·因幡_04.png.md)能够快速抽干目标的力量值直接使目标变成脆皮（秋穰子的大招对幻想有效无论是释放还是作用），可以在对面选出[](./文件-THD2蕾米莉亚·斯卡雷特.png.md)克制大兔子时点出来反克制，也可以克制对面的冲脸肉核，需要注意的是秋穰子的[](./文件-THD2秋穰子_03.png.md)光环无法作用于幻象，容易暴露大兔子的真身
```

```
 [](./文件-THD2蕾米莉亚·斯卡雷特.png.md)：稳定的先手可以让大小姐神枪必中，[](./文件-THD2秋穰子_04.png.md)配合[](./文件-THD2蕾米莉亚·斯卡雷特_04.png.md)可以让大小姐冲阵更安全的同时更快削弱周围人的血量，在对面选出[](./文件-THD2铃仙·优昙华院·因幡.png.md)时，也可以反抢一手秋穰子来配合大小姐克制
```

秋穰子不管打4.5号位都是很出色的。在对局中要尽量去发挥其奶妈、救人、续航、团战保护阵型的优势。秋穰子不需要无脑冲（当然她的身板也不会像其他辅助一样突然歇逼），只需要跟着队友一起，提供光环，不断的给苹果和小车就行。所以也不需要太多的装备，非要说质变装备，那必然是万宝槌。  
  

出门装方面，可以出一个御币加一张爱人卡，同时也可以拉一张果篮的卷轴并锁定合成，御币提供续航，觉得压力不是很大可以解锁卷轴合成果篮，爱人卡当然是防止自己废物大哥暴毙，之后的出装可以先出一个礼物，肥一点可以出礼服，之后直接奔着万宝槌去就完事了，谁不喜欢一下扔出去10个苹果的快感？之后就可以再出一些常规的辅助装备了，像什么星星，思兼智慧，蛤帽之类的，或者是镜子，十字架，吹风，推推，毒瘤点还能出核棒，cd转的快。
