东方非想天则 | 原作游戏解锁方式

-  **剧情模式** (Story Mode)有3个可用角色（一开始只有1个可用角色，有2个角色需解锁）
-  **对战模式** (Adcade Mode/Vs Com/Vs Player/Vs Network)有20个可用角色（一开始只有7个可用角色，有2个角色需解锁，与东方绯想天整合之后可再开启11个角色）  


  
各锁定角色解锁方式:  

  

博丽灵梦
:  **剧情模式** ：不可使用  
 **对战模式** ：初期可使用  

雾雨魔理沙
:  **剧情模式** ：不可使用  
 **对战模式** ：初期可使用  

东风谷早苗
:  **剧情模式** ：初期可使用  
 **对战模式** ：初期可使用  

琪露诺
:  **剧情模式** ：东风谷早苗剧情模式完成后可使用  
 **对战模式** ：初期可使用  

红美铃
:  **剧情模式** ：东风谷早苗和琪露诺剧情模式完成后可使用  
 **对战模式** ：初期可使用  

灵乌路空
:  **剧情模式** ：不可使用  
 **对战模式** ：东风谷早苗剧情模式完成后可使用  

洩矢诹访子
:  **剧情模式** ：不可使用  
 **对战模式** ：东风谷早苗剧情模式完成后可使用  

爱丽丝·玛格特洛依德
:  **剧情模式** ：不可使用  
 **对战模式** ：东风谷早苗和琪露诺剧情模式完成后可使用  

帕秋莉·诺蕾姬
:  **剧情模式** ：不可使用  
 **对战模式** ：东风谷早苗，琪露诺和红美铃剧情模式完成后可使用  

伊吹萃香

 **对战模式** ：和绯想天联动后可使用  

八云紫
