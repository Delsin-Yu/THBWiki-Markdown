## 基本介绍
### 个人介绍
笔名、昵称
:  **JynX** 
: 其实祂原来的笔名是JinX，多次修改后确定为JynX[^cite_note-1]&#91;[JynX访谈（2015）&#93; (未找到链接)
能力
: 
: 有时可以完美掰开一次性筷子[&#91;Trick Nostalgie公式站&#93;](http://mugentrick.tubakurame.com/)
: 制作游戏程度的能力[&#91;JynX的推特主页（2018）&#93;](https://twitter.com/mugen_Jinx/status/963699487238574080)
: 全年都感到困倦程度的能力[&#91;JynX的推特主页（2020）&#93;](https://twitter.com/kouki_mono/status/1226339188741558272)
喜好
: 

-  **饮食** 

: 酒中最喜欢喝啤酒，品牌酒中喜欢喝Dunkel和Kölsch，此外还喜欢喝黑咖啡和不加糖的红茶。&#91;JynX访谈（2015）Q41、JynX访谈（2015）Q41&#93; (未找到链接)
: 对于食物，JynX喜欢生姜烧肉和饺子。&#91;JynX访谈（2015）#Q41、Trick Nostalgie公式站&#93; (未找到链接)

-  **动物** 

: 喜欢鸟类、爬行动物和两栖动物,[&#91;JynX访谈（2017）Q128&#93;](http://lenen.shoutwiki.com/wiki/Len'en_Wiki:Interview_with_JynX_(2017))并养有两只乌龟，分别叫作ジロー和チビ。[{{{2}}}]

: 此外，JynX还对龙感兴趣。[&#91;JynX访谈（2017）Q128&#93;](http://lenen.shoutwiki.com/wiki/Len'en_Wiki:Interview_with_JynX_(2017))

-  **游戏音乐** 

: 除了ZUN以外，JynX还喜欢诸如椙山浩一、下村阳子、樱庭统、光田康典等等的电子游戏作曲家。[&#91;JynX访谈（2017）Q131&#93;](http://lenen.shoutwiki.com/wiki/Len'en_Wiki:Interview_with_JynX_(2017))

## 其他资料
- 对于自己作的连缘音乐，JynX喜欢[In This World　～ Monochrome_eye](./In_this_World_～_Monochrome_eye.md)、[埴轮相扑　～ End_of_immolation](./埴轮相扑_～_End_of_immolation.md)、[在深邃徒云之下　～ Mow_Down!](./在深邃徒云之下_～_Mow_Down!.md)、[幽暗海峡启航](./幽暗海峡启航.md)、[梦中遗忘的神圣萤火](./梦中遗忘的神圣萤火.md)和[显界Overhaul](./显界Overhaul.md)。&#91;JynX访谈（2015）Q47&#93; (未找到链接)
- 对于自己制作的符牌，鸟之目「黑之瞳」给JynX留下了深刻的印象，祂在如何用弹幕表示黑色的眼睛这方面思考了很多。&#91;JynX访谈（2015）Q46&#93; (未找到链接)
- JynX表示如果自己被神隐到[无现里](./无现里.md)的话，祂会四处游历，喝遍当地美酒，最终变成一个妖怪。[&#91;JynX访谈（2017）Q133&#93;](http://lenen.shoutwiki.com/wiki/Len'en_Wiki:Interview_with_JynX_(2017))
- 在高中时，JynX曾想成为物理学家。[&#91;JynX访谈（2017）Q132&#93;](http://lenen.shoutwiki.com/wiki/Len'en_Wiki:Interview_with_JynX_(2017))
- JynX表示自己有想制作连缘×东方的作品的意愿，但是会非常惶恐。&#91;JynX访谈（2015）Q39&#93; (未找到链接)
- ZUN曾说过“东方Project已成为我毕生的事业。”，JynX对此深表同感，只不过可能会做连缘世界观下薮雨和玄鸟以外的主角的游戏。&#91;JynX访谈（2015）Q50、JynX访谈（2017）Q142&#93; (未找到链接)


[^cite_note-1]: 蚁䴕（学名：Jynx torquilla），即JynX的鸟之名
