メリーがこの世を去ってから一ヶ月と半分が過ぎた。かつての日常を奪われた私は、狭い部屋の中で壊れた機械のように生きていた。  

あの電話が鳴る瞬間までは。  

それは法的にも物理的にもすでに存在しないはずの、ありえない「彼女」からの着信だった。  

色のついた非日常が帰ってくる。  

止まっていた時間が回り始める。  

震え続ける端末に私は手を伸ばす。  

その先に待つものが、一体何であるのかも知らないまま。
