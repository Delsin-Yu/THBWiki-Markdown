# 东方心绮楼/角色解锁

<!-- source html: G:\repos\THBWiki-Markdown-Builder\THBWikiMarkdown\Temp\main\9\90\ns0%3A%E4%B8%9C%E6%96%B9%E5%BF%83%E7%BB%AE%E6%A5%BC%2F%E8%A7%92%E8%89%B2%E8%A7%A3%E9%94%81.html -->

东方心绮楼 | 原作游戏解锁方式

-  **剧情模式** （Story Mode）有10个可用角色（一开始只有2个可用角色，其他角色需解锁）
-  **对战模式** （VS Com/VS Player/VS Network）也有10个可用角色（一开始只有8个可用角色，其他角色需解锁）

  
各个锁定角色的解锁方式：
  

博丽灵梦
:  **剧情模式** ：初期可使用  
 **对战模式** ：初期可使用

雾雨魔理沙
:  **剧情模式** ：初期可使用  
 **对战模式** ：初期可使用

圣白莲
:  **剧情模式** ：博丽灵梦和雾雨魔理沙的剧情模式完成后解锁  
 **对战模式** ：初期可使用

丰聪耳神子
:  **剧情模式** ：博丽灵梦和雾雨魔理沙的剧情模式完成后解锁  
 **对战模式** ：初期可使用

云山&amp;云居一轮
:  **剧情模式** ：博丽灵梦，雾雨魔理沙，圣白莲和丰聪耳神子的剧情模式完成后解锁  
 **对战模式** ：初期可使用

物部布都
:  **剧情模式** ：博丽灵梦，雾雨魔理沙，圣白莲和丰聪耳神子的剧情模式完成后解锁  
 **对战模式** ：初期可使用

河城荷取
:  **剧情模式** ：博丽灵梦，雾雨魔理沙，圣白莲和丰聪耳神子的剧情模式完成后解锁  
 **对战模式** ：初期可使用

古明地恋
:  **剧情模式** ：博丽灵梦，雾雨魔理沙，圣白莲和丰聪耳神子的剧情模式完成后解锁  
 **对战模式** ：初期可使用

二岩猯藏
:  **剧情模式** ：博丽灵梦，雾雨魔理沙，圣白莲，丰聪耳神子，云山&amp;云居一轮，物部布都，河城荷取和古明地恋的剧情模式完成后解锁  
 **对战模式** ：完成二岩猯藏的剧情模式后可使用

秦心
:  **剧情模式** ：以上全部人物剧情模式完成后解锁  
 **对战模式** ：v1.20版本之后剧情模式全部完成（即完成秦心的剧情模式）后可使用
