## 道具
蓝点
: 
星星
: 击破任何敌人都会掉落，贴近道中杂鱼击破会掉出额外一个星星，贴近boss和大蝴蝶射击会持续获得星星（有暗色圈提示）。E-L难度分别获得不同数量的最大得点，分别是8/10/15/20，ex为30，增加1点星座槽，获得1章节道具数。
残机碎片
: 星座槽攒满（收集333个星星）后获得一个，固定每三个残机碎片奖励一个残机。勇者模式下，收集星星会奖励bomb碎片而不是残机碎片。
bomb碎片
: 在章节中完成nomiss后获得一个，固定每三个bomb碎片奖励一个bomb。
收点线
: 屏幕左边的绿色箭头，顾名思义是收取道具的高度，击破敌人后会降低，降低的高度基于敌人的种类&amp;血量。
残机道具
: 击破三面、五面、ex道中boss固定掉落1残机道具

  
（勇者模式下会掉落1bomb道具）
  

bomb道具
: 击破二面、四面、六面道中boss固定掉落1bomb道具

## 星座槽
  
位于屏幕右边分数下，显示目前星等以及获得残机，增加倍率。星座槽所显示的星座与人物称号成对应关系，即灵梦对应处女座，魔理沙对应射手座，早苗对应蛇夫座。
  

星等
: 每收集333个星星之后星座槽攒满一次，同时提升一等星（注：ex关为100个星星），一共分为6/5/4/3/2/1六个等级[^cite_note-1]。每一等级对应相应倍率，分别对应1.0/1.6/2.2/2.8/3.4/4+（在一等星之后每攒满一次星座槽增加0.05倍率）。miss后降低一个星等，倍率降低至星等对应倍率。
Last Word
: 每一面终符结束后开启的额外符卡，在该SC中无法使用bomb，中弹不损失残机。（六面除外，6boss的LW算作通常符卡，可以丢b，中弹会损失残机。）在星座槽左下角的小数字对应LW开启需要的星星数量，够数后会在星座槽上提示 **LastWord Standby** 。1至6面开启LW需要的星星数目为777/777/1333/1333/1777/666（在圣经中，666是恶魔的象征）
: 这里需要注意的是，击破boss掉落的星星会出现一定的显示bug，在下一次死亡后才会计入LW的统计中，原因不明

## 分数系统
蓝点
: 在收点线以上收取，获得最大得点x倍率的分数
结算点
: 和绀珠传的结算点一样，每个结算点奖励一定的分数，分数奖励公式为【该章节获得星星数x结算时倍率x结算时最大得点x击破率÷（miss次数+1）】
最大得点
: 不同难度初始最大得点不同，e/n/h/l分别为10000/15000/20000/25000，miss后会降低最大得点，降低的数额为当前最大得点的10%
擦弹
: 每5个擦弹获得10最大得点
SCB（符卡奖励分数）
: 收取符卡后获得的分数，

  
SC起始分數為(最大得點*得分倍率*120)
SC額外得分起始為(最大得點*得分倍率*星點總數*0.777),擦彈不計
爾後加分數依星點和擦彈數計算
SC額外得分的擦彈分約為最大值的千分之一[^cite_note-2]
  

通关得分
: 每一面结束后的固定值得分，每一面分数不同，不同难度分数不同。

星星
: 贴脸射击得到的星星会自动回收，每个上述星星获得E-L难度分别获得不同数量的最大得点，分别是8/10/15/20，ex为30。
消弹绿点
: 符卡结束自动全屏消弹或发动bomb在bomb效果范围内消弹，把子弹变成绿点。每10个绿点增加10最大得点，注意激光等特殊弹幕的消弹机理不同。
资源溢出
: 当残机&gt;=8+2/3时吃残机碎片或spell&gt;8+2/3时得到雷碎片，分数会增加最大得点*得分倍率*7.77的数值；残机为8（碎片不管）吃到完整的残机或类似情况的雷时，得到三倍上述数值的分数。[^cite_note-3]
