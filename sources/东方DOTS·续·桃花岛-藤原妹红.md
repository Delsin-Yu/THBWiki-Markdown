```
[](./文件-THD2格斗扫把.png.md) [](./文件-THD2纳税阴阳玉.png.md)
```

  
  

  
  
尽早做出钱玉进野区，加点注意多加w再进野，w技能带来的攻速被动伤害完全比e技能刷新q技能撞野来的快
  

### 核心装备
```
[](./文件-THD2格斗扫把.png.md) [](./文件-THD2念缚灵的船勾.png.md) [](./文件-THD2天界之桃.png.md) [](./文件-THD2龙星.png.md)
```

  
  

  
  
由于藤原妹红的是近战少女且q技能会伤害到自己，靠受到伤害叠加层数的装备会非常适合她，当前环境打团龙星是必备的，因为[](./文件-THD2念缚灵的船勾.png.md)和[](./文件-THD2天界之桃.png.md)buff会存在一定时间，可以通过不断地q地板然后开e技能刷新血量和q技能叠层数，这样进场后便可以有健康的血量量和无风险装备叠加最大层数，前提是这样操作的时候不要被对面知道你在哪（前车之鉴，神装第一波出山前叠buff被对方直接a蛋打出GG）
