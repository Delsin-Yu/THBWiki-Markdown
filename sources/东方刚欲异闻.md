## 游戏系统
  
本作的定位是动作游戏，无论是宣传还是实际都是如此，并非格斗游戏。  

但由于没有道中，没有平台跳跃与机关解谜要素，实际上表现和之前黄昏格斗作的故事模式有些相似。[^cite_note-1]  

每个自机各有故事模式，没有对战模式。  

不同自机的关卡数并不一致，暂停界面选择retry或疮痍后选择续关，会从当前关卡的开头重新挑战（无对话）。  

无replay系统，只会记录通关记录与分数。
  

- 相比之前版本的主要更新

1.02版本后
: 比起只能按顺序完整通关，可以选关挑战了；
1.14版本后
: 增加了Easy难度；
: Easy难度每次续关都会增加一个生命球，最多为9个生命球（18格血）；
: 增加了饕餮尤魔作为可用角色；
: 增加了成就系统；
: 增加了刚欲挑战模式。

- 游戏难度

  
本作分为Easy、Normal和Hard三个难度，在游戏主体内容上，三个难度基本没有差异，其主要差别主要体现在以下方面：
  

自机血量不同
: Easy和Normal难度每个生命球代表两格血，Hard难度每个生命球只代表一格血；
续关效果不同
: Easy难度续关时会增加生命球，Normal和Hard难度续关时并不会增加生命球；
关卡效果不同
: 在“旧灼热地狱”对战灵乌路空的关卡时，Easy和Normal难度触碰火墙不会造成伤害，Hard难度会造成伤害。

- [各角色机体说明](./东方刚欲异闻-机体说明.md)

  
本作共七个自机，每个自机都有独特的攻击模式。  

  

- [各角色解锁方式](./东方刚欲异闻-角色解锁.md)

  
初始可用博丽灵梦与雾雨魔理沙两个自机，使用自机初次通关故事模式后，会解锁新的自机。任意难度解锁过的自机可在所有难度中使用。  

  

- [操作练习](./东方刚欲异闻-操作练习.md)

  
本作也是东方系列中继PS4版[东方深秘录](./东方深秘录.md)后，又一款拥有操作练习教程关的官方游戏。
  

- [水系统](./东方刚欲异闻-水.md)

  
本作特色的系统为水系统。擦弹（键盘默认Z键）与攻击（键盘默认X键）均可以消除弹幕且不受伤害。  

擦弹消除的弹幕会转化为水驻留在场中，此时按吸收水键（键盘默认C键）可以吸掉水填充水槽，水槽填充100％后再次按吸收水键可以使用强力的解放攻击。  

水会受地形影响，流出屏幕或留在地形凹处，并受到角色运动影响。部分场景中水会有自机接触减速等特殊效果。  

攻击可以消耗体力用擦弹打断，因此自机操作比较灵活。  

BOSS使用符卡时有蓝色魔法阵与红色魔法阵两种背景，红色魔法阵背景时受到多次攻击会破盾，全屏消弹，并进入一段时间的瘫痪状态。  

瘫痪状态中自机对BOSS造成的伤害翻倍。有护盾的符卡中，破盾优先消耗掉护盾，BOSS直接重新释放符卡。对战开始时BOSS有一定时间的无敌。
