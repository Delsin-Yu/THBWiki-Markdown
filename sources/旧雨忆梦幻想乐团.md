## 其他信息
- 该社团的看板娘“娇弱酱”，是一个留有黄色长发，穿着类似制服的服饰的美少女。其原型为该社团主催穆然皎若的女装照。

: 在旧雨忆梦幻想乐团于bilibili发布的第一个视频“《旧雨忆梦幻想乐团成立》，看板娘娇弱酱主题曲！【东方风合作曲】”中，看板娘娇弱酱的背景故事首次亮相。
: 现实世界的皎若先生在通宵玩Galgame的时候猝死过去从而一不小心穿越进了游戏中变成了女主角“娇弱酱”，
: 因为熟悉剧情所以娇弱酱很快找到了各种各样的机遇登上了城市的舞台，成为了“雨之偶像 娇弱酱”，那么接下来娇弱酱会遇到什么样的挑战呢？敬请期待！

- 该社团的第一部东方风自作曲集的前身为穆然皎若独立创作的曲集《东方山海传》。

: 在社团成立后，正式成为东方风自作曲集&amp;设定集，并在2023-08-27的0:00准时发布于bilibili。
: 于2023-09-23，在bilibili上发布正式版。

- 该社团的作品被归纳为“忆梦宇宙”当中，其中，东方山海传为YM01，东方革金机为YM02。

: 但实际上，该制作组的作品是与邪导桃魔制作组的同人作品为同一世界观下共存的。
: 因此，在作品的对话中会略微提到其他同人作的异变。

- 该制作组的第二部曲集&amp;设定集作品《东方革金机 ~ The Golden Age Will Return Again》，最初是鸢鸢鸢鸢子独立创作并发布于bilibili专栏的东方二次设定集。

: 在2023-09-23，正式宣布成为旧雨忆梦幻想乐团的下一部曲集&amp;设定集作品。
