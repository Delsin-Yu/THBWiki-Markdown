## 评论
```
ある日、慧音のもとを訪れた魔理沙は昔の新聞記事を彼女に見せて頼み込みます。
「消して欲しい過去（れきし）がある」
それは、未だに彼女を捕らえる心の傷。食べていくために、そして、ある一途な想いゆえに犯してしまった過ち。
あなたにも、未だに消えない心の傷はありませんか？
```

```
[蜜瓜/社团评论](https://www.melonbooks.co.jp/detail/detail.php?product_id=218597)：ある日、慧音のもとを訪れた魔理沙。
慧音の「歴史を食べる能力」を頼って、彼女は「消して欲しい過去(れきし)」がある」と告げて己の過去の独白を始める。
それは、まだ幼い彼女が自立し、その日を生きるために、そして、小さくも大切な想いを育むための過ちであった。はたして、慧音は魔理沙を救えるのだろうか。
```
