## 评论

  
とりあえず成し遂げたかったこと  

  

　私は以前にも東方ドラムンで合同企画がされていたことを知っています。 結局主催の多忙により企画倒れになったらしいですけど。 その断片はググると多分出てきます。  

　こんな企画があったので,ドラムン好きの私がやらなければ,この縮小気味の界隈で誰がやる?ってことで企画したのが始まり。 ...まあぶっちゃけただ単に自分が他人のアレンジが聴きたかっただけですけどね。  

　ついでに曲の話ですが,ドラムンとドリルンと銘打ってるので,それとなく「グレー」なアレンジを作りたかったのです。 Anonymous Radio Stationなんか違法サンプリングで絶版になったCDからそのまま使ってる上,The Flashbulbの真っ赤なジャケのやつからサンプリングしたりしてます。 あとドラムンに関してはBootleg。 ...怒られたら消します。  

　あとは私みたいなもの好きのためにDLリンクを切らせないようにしたいなぁと。 もうだいぶ昔に企画された東方アレンジ合同でリンクが切れてたのを,わざわざマスタリングを担当していた方にお願いして頂いた過去がありますので...(再調整さん本当にありがとうございました。)
