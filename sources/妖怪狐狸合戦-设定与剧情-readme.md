=================<br><br>◆操作方法<br><br>  ショットキー  …通常ショット<br>  ボムキー      …スペルカード発動<br>  ユーザーキー　…化け波動発動（化け力が100%の場合のみ。発動時の方向キー入力で若干挙動が変化）<br><br>　キーコンフィグは弾幕風ph3本体に添付されている"config.exe"から行って下さい。<br><br><br><br>◆アイテムについて<br><br>　[点]　…　点アイテム。得点が加算されます。<br>　[Ｐ]　…　Pアイテム。化け力が1%回復します。<br>　[化]　…　[点]、[Ｐ]に化け波動を当てるとこのアイテムに変換されます。<br>            戦力と同値の得点加算、化け力を1%回復。更に戦力の値も増加。<br>            また、エクステンドゲージ（後述）が少し増加します。<br><br>　(化)　…　化アイテム(小)。<br>            敵弾を化け波動やスペルカードに当て変化させた場合に発生します。<br>            敵を倒した時にも少しだけ出現します。<br>            化け力回復量は[化]の1/5、得点は1/100増加します。<br>　(点)　…　点アイテム(小)。得点が少し加算されます。<br>            スペルブレイク時にのみ出現するレアアイテム。<br><br>　[Ｓ]　…　スペルカード。スペルカードが1つ増加します。<br>　[Ｌ]　…　ライフ。[化]40個ぶんだけエクステンドゲージが増加します。<br><br><br></div></td><td class="tt-zh" lang="zh"><div class="poem">===============  关于本游戏  =================<br><br>◆操作方法<br><br>  射击键        …通常射击<br>  炸弹键        …发动符卡<br>  自定义键    　…发动妖怪波动（仅限妖力100%时。按照发动时输入的方向键有所变化）<br><br>　键位设定请在弹幕风ph3本体附带的"config.exe"中进行。<br><br><br><br>◆关于道具<br><br>　[点]　…　点道具。增加得分。<br>　[Ｐ]　…　P道具。回复1%妖力。<br>　[化]　…　[点]、[Ｐ]受妖怪波动影响会变为此道具。<br>            增加与战力同值的得分，回复1%妖力。战力值也会增加。<br>            此外，残机槽（后述）也会增加一点。<br><br>　(化)　…　化道具(小)。<br>            敌弹受妖怪波动或符卡影响会变为此道具。<br>            打倒敌人时也会出现一些。<br>            妖力回复量为[化]的1/5，得分增加1/100。<br>　(点)　…　点道具(小)。稍微增加得分。<br>            仅在突破符卡时出现的稀有道具。<br><br>　[Ｓ]　…　符卡。增加一张符卡。<br>　[Ｌ]　…　生命。残机槽增加大约40个[化]的程度。<br><br></div></td></tr><tr class="tt-content" id="=-3" data-pos="&#91;&quot;=&quot;,3&#93;"><td class="tt-ja" lang="ja"><div class="poem">◆化け力、化け波動<br><br>　化け力≒妖精大戦争の氷ゲージ<br>　任意でチャージする事ができず、時間経過・アイテム取得などによって増加します。<br>　100%になると自機の周囲にエフェクトが表示されます。<br><br><br>　化け力が100%になるとユーザーキーで化け波動を放つ事ができます。<br>　ボムと似た性能ですが、ダメージ判定はなく、<br>　喰らいボム待ち受け時間中に使用しても発動せず、そのまま撃墜されてしまいます。<br><br>　化け波動を使うと、<br>　範囲内の敵弾を(化)アイテムに、更に[点]や[Ｐ]アイテムを[化]アイテムに変化させます。<br><br>　敵弾をアイテム化させた場合の化け力回復量にはリミッターがかかります（基本的に50%まで）<br>　しかし、それ以外のアイテムについての制限はないので、<br>　敵弾変化＋アイテム変化を同時に行う事により、即座に100%まで回復させる事も可能です。<br><br>　纏めると…<br>　・化け波動はリチャージできるボム。ただしダメージはなく、喰らいボムもできない。<br>　・敵弾やアイテムが沢山ある所にぶっぱすれば美味しい。<br><br><br></div></td><td class="tt-zh" lang="zh"><div class="poem">◆妖力、妖怪波动<br><br>　妖力≈妖精大战争的冰槽<br>　但并不能任意蓄力，而只会在时间经过、取得道具等时候增加。<br>　达到100%自机周围会有效果表示。<br><br><br>　妖力达到100%时，按自定义键可以释放妖怪波动。<br>　虽然性能和Bomb很像，但没有伤害判定，<br>　在被弹输入时间内也不能发动，而只会被击坠。<br><br>　使用妖怪波动后，<br>　范围内的敌弹会变成(化)道具，[点]及[Ｐ]道具会变成[化]道具。<br><br>　将敌弹变为道具时，妖力回复量有上限（基本不超过50%）<br>　但除此之外的道具没有限制，<br>　所以在敌弹变化+道具变化同时发生时，可以立刻回复至100%。<br><br>　总结一下……<br>　・妖怪波动是可以重复使用的Bomb。但没有伤害，也不能决死。<br>　・在敌弹、道具成堆的地方发动很爽。<br><br><br></div></td></tr><tr class="poem">◆戦力、コンボゲージ<br><br>　戦力はいわゆる最大得点。<br>　主に[化]アイテム回収により稼ぐことができ、自機が撃墜されると戦力の10%が失われます。<br>　基本的に得点稼ぎ用ですが、稼いでおくと最終戦が少し楽になります。<br><br>　戦力表示欄の裏にある増減するゲージはコンボゲージ。<br>　空になると戦力が徐々に減っていきます。<br>　[化]アイテム取得で満タンに、それ以外のアイテムを取得すると少し回復します。<br>　また、ボス戦時に減る事はありません。<br><br><br></div></td><td
