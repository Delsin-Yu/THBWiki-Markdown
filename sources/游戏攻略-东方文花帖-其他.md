# 游戏攻略/东方文花帖/其他

<!-- source html: G:\repos\THBWiki-Markdown-Builder\THBWikiMarkdown\Temp\main\8\82\ns0%3A%E6%B8%B8%E6%88%8F%E6%94%BB%E7%95%A5%2F%E4%B8%9C%E6%96%B9%E6%96%87%E8%8A%B1%E5%B8%96%2F%E5%85%B6%E4%BB%96.html -->



- 本作的相机无法以一定角度倾斜
- 与永夜抄类似，文花贴也有可以引boss的机制，善用此机制会减小部分关卡的拍照难度（例如4-1,7-7等）
- 在超低速模式下，只松开低速键可以直接远拍
- 若该关卡已通关，但是拍到的分数较低，则文对该关卡的评价为：

うーん。枚数は十分撮れたんだけど……
もうちょっと使えそうな写真が撮りたいわね。
この程度の写真じゃ面白い記事は書けないわ。
（翻译：嗯……虽然已经拍了很多的照片，
但还是想快点拍些可以用的出来，
这种程度是无法撰写有趣的新闻的。）
