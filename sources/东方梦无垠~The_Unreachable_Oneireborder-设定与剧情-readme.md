___

  
&#160;· 1.前言  

	首先非常感谢您下载和游玩由实界制作的东方梦无垠 ~ the Unreachable Oneiroborder 体验版pre1。  

	我是实界的主催云绝万里。  

	这是一部在梦与现实纠葛不清的幻想乡中，一群本应轻松悠闲的少女展开的无比激烈的弹幕射击游戏。  

	游戏的难度跨度会非常大，easy会简单过头，lunatic会直逼绀珠传，对新手会很友好，一直按住X键也能通关（不  

	虽然我们已经尽力了，但是作为新人，缺乏经验，难免会有做的不好的地方，大家只管提意见就是。  

  

___

  
&#160;· 2.运行方法  

	双击game目录下的LuaSTGPlus.exe即可启动游戏  

  

___

  
&#160;· 2.5 注意事项  

	请确保你的电脑剩余&gt;1200MB的内存  

	请确保你的电脑的分辨率至少为1920*1080，如果低于这个值那么游戏很可能无法运行。  

	在满足以上条件下如果设置&gt;系统&gt;显示&gt;缩放与布局中的比例不为100%可能无法以窗口模式运行游戏（窗口会消失）。  

	可以尝试在任务栏点选该窗口并按下Ctrl+Enter进入全屏，请去设置中选择更小的分辨率。  

	如果还不奏效，请删除游戏根目录下Library\setting文件再重启游戏。  

	游戏暂时不支持储存rep，在之后的版本会跟上。  

	游戏目前有三个机体，梦B在之后的版本会跟上。

	也可以直接在围观群里发言。  

  

___

  
&#160;· 5.如果想要加入我们  

	很开心，因为我们人力急缺。但在那之前请浏览一下我们给出的要求：  

	· 平均下来每周至少能抽出两天整的时间用来制作游戏。  

	· 善于交流，懂得合理使用通讯工具而不是不管在不在线都不吱声  

	· 拥有一定程度的学习能力，即使是被委托做职责外的事情也敢于尝试  

	· 有热情和决心把手头的事情做好做完（免疫鸽瘟  

	· 基本的规划能力  

	我们需要：  

	· lstg3D背景制作  

	· Boss或者道中弹幕制作  

	· 宣传  

	· 文案  

	如果自身条件满足并已经考虑清楚，请联系云绝万里。
