矢田寺成美（官方角色，新作） - 新作角色。魔法森林的地藏像，因森林的魔力化为魔法使
### 角色介绍
  
矢田寺成美是东方Project系列中的官方角色，首次登场于[东方天空璋](./东方天空璋.md)，担任天空璋的四面Boss。  

她是在[魔法森林](./魔法森林.md)的魔力驱动下得到生命的地藏像，属于一种[魔法使](./魔法使.md)。  

她在官方游戏与出版物中登场次数不多，而且跟剧情没有什么联系。
  

#### 生活状况
  
过去被安置在[魔法森林](./魔法森林.md)中，现在仍然居住在此。  

是个家里蹲，即使发生[异变](./东方天空璋.md)也不愿离开森林去调查。
  

- 然而，她的一项工作却是守望[赛之河原](./赛之河原.md)的垒石不让它们倒塌&#91;黑市&#93; (未找到链接)，并且她后来参加了在赛之河原举办的[黑市](./弹幕狂们的黑市.md)。

#### 角色能力
  
成美拥有 **使用魔法程度的能力（生命操作）** 。
  

- 运用这项能力，成美能够操纵生命力。&#91;天空璋&#93; (未找到链接)
- 此外，她认为子弹也是活着的&#91;GoU&#93; (未找到链接)，[宇佐见堇子](./宇佐见堇子.md)则认为她把生命体包裹在弹幕里&#91;噩梦日记&#93; (未找到链接)。

#### 外貌特征
  
成美有着一头黑色及腰长发，脑后系着两条麻花辫。头戴一顶 **遍路笠** ，在天空璋中上面写着“迷故三界城/悟故十方空/本来东西无/何处有南北/同行二人/यु”。斗笠上覆盖着残雪里面内衬白布，边缘打着褶子。  

她有着红色的眼瞳，长着长耳垂。  

她身穿灰色大衣，纽扣为红色，左肩披着红色披巾，在天空璋中有黑色的汉字“毕竟成佛”。  

她脚穿红绳短袜和黑色木屐。  

作为地藏像，她的立绘总是双手合十。

#### 人际关系
  
成美与[雾雨魔理沙](./雾雨魔理沙.md)同住在[魔法森林](./魔法森林.md)，很早就相互认识了。她被魔理沙叫作“成子”。
  

#### 登场剧情
  
本是长时间镇守在[魔法森林](./魔法森林.md)中的地藏像，受森林魔力的影响而得到生命，变成了魔法使。
  
  
在[东方天空璋](./东方天空璋.md)中，[丁礼田舞](./丁礼田舞.md)最初选择了成美作为新伙伴，在前去见成美时被自机追逐，于是在成美的背后开了一扇门来涌出魔力。成美感受到了魔力涌出，以为是异变的影响，却没发现背后出现了门，并作为四面Boss与自机打了一架，被打败。之后自机发现并跳进成美背后的门（琪露诺线则是琪露诺发现并跳进自己背后的门）。
  
  
在[新东方香霖堂第4话](./东方香霖堂-东方外来韦编第4话.md)中，在去[人类村落](./人类村落.md)的路上因为太冷而到[香霖堂](./香霖堂.md)休息，遇到[森近霖之助](./森近霖之助.md)、[雾雨魔理沙](./雾雨魔理沙.md)、[宇佐见堇子](./宇佐见堇子.md)。  

之后被堇子一顿乱摸，之后言语上反击了堇子，但很快与堇子和魔理沙愉快地聊起来了。  

之后，追问魔理沙异变为什么不结束，在听她讲完异变的详细经过后，对[摩多罗隐岐奈](./摩多罗隐岐奈.md)感到畏惧。

#### 设定文档
　○４面BOSS　　垂迹于森林的魔法地藏  
　　矢田寺 成美  
　　Yatadera Narumi  
  
　　种族：魔法使（地藏）  
　　能力：使用魔法程度的能力（生命操作）  
  
  
　　她是原本是安置在魔法森林的地藏，因为森林的魔力而得到了生命的生命体。  
　　虽然魔法森林受困于不合季节的雪，但她原本就是个家里蹲所以并没有离开森林进行调查的想法。  
  
　　在魔法森林里面有着能够把存在本身进行变化的魔力。  
　　虽然说她是地藏，却并不是地藏菩萨本身，而是原本是地藏像，也就是石像。  
　　原本就是石像获得了生命一样，像是魔像Golem一样的存在。  
  
　　她对魔力很敏感，已经察觉到了自己背上正在涌出异常的魔力。  
　　但是，她觉得这应该是四季异变的影响，却完全没有察觉到  
　　是这现象本身引起了异变。  
  
　　从魔理沙那里听说了隐岐奈的事情，想到了一个秘策，然后托付给魔理沙的正是她。

### 宇佐见的魔法书
魔符「作宠物的巨大弹生命体」 (未找到链接)使用者：使用者自述大家都在努力地亲自发射弹幕，但那样可实在是谈不上聪明，对我来说子弹也是活着的，就用这可爱的巨型火球来吓大家一跳！可爱的宠物定会受到大家的无比喜爱！博丽灵梦（7）弹幕的落点太靠近观众席还真让人捏了一把汗，但发现火焰只是假货后就安心了。雾雨魔理沙（6）那个巨大的球弹真的是活着的吗？怎么看都只是本体在后面操控吧……宇佐见堇子（8）从近距离观赏的话烟花也十分有魄力，但这种烟花不是只能让人感到害怕吗。东风谷早苗（5）活着的巨大球弹，真的很可爱吗？这种价值观有点难以接受。魂魄妖梦（8）真、真是可爱，不会说话的宠物就是好。综合评价（7）巨大的火球降落到观众席之中的演出在惊人的角度上确实很不错，同时也有考虑安全性的样子所以评价很高。但是，留下了“这真的是烟花的乐趣吗”的疑问。地藏「业火救济」 (未找到链接)使用者：使用者自述嘻嘻，刚才只是为引出这块玉所抛的砖而已，我的宠物巨大弹已经向地面撒出了大量的生命弹，前无古人，从观众席喷涌而出的烟花！没有不为之惊叹的人！博丽灵梦（0）这、这也太危险了！雾雨魔理沙（3）原来如此，从观众席中涌出的火弹吗……诶，危险！宇佐见堇子（9）刚刚那个宠物撒出的东西，原来是火种啊，居然连评审员都欺骗，真是不错的想法。十六夜咲夜（5）刚刚的是假货，这次的火焰却是真家伙，还真敢做呀。东风谷早苗（5）仔细想一想……从地面发射的火弹一点也不让人意外，不就是传统的烟花吗……综合评价（4）地面突然涌出火焰，导致会场都陷入了恐慌。虽然她吓人的目的达到了，却没人在意烟花了。可能是她并没有好好理解到烟花大会真正的有趣之处。

#### Comment from ZUN
　石头色的衣服配上红色的围巾，她的服装看上去就像地藏一样。成美戴着斗笠的形象，源自传说故事中的斗笠地藏，但她戴着的其实是进行八十八所巡礼的行者所戴的那种斗笠。不知道她原本就是八十八所巡礼途中的一尊地藏，还是说她向往着那些进行巡礼的苦行者，又或者说她其实是已经完成了多次巡礼的猛士……说起来，不知现在的人还会不会造地藏像呢？现在许多人逐渐远离信仰，感觉今后就只剩日渐风化最终归于尘土的地藏，以及被视作文物特意保存下来的地藏了。

: 

对话的时候气泡上的名字是写作「成美」，战斗的时候体力槽下面写的却是「Naruko」，魔理沙对其的称呼也是「成子」。
ZUN
: 

这么说来我觉得「成美」这个名字才是对的吧。起初是叫作「成子」，后面改成「成美」了，但是有一部分字样没改全呢。
——
: 

有人觉得那大概是魔理沙起的绰号。
ZUN
: 

这么理解也顺，叫哪个名字都无所谓不过虽说无所谓但是「〇〇子」这样的名字已经很多了所以总觉得有点微妙。气泡上的名字是最先修改的部分，所以那上面的名字一定是对的。起初大概是以成子木娃娃[^cite_note-1]为印象起的这个名字的。

: 

矢田寺是一处有名的祭拜地藏像的地方，不过那些有名的地藏像都挺多灾多难的啊。会被人裹上棉被之类的。

: 

是挺多灾多难的。会有人给它们化妆或是会被扔到河里，还会被拔起来转圈圈。不过毕竟是建在有很多人烟的地方，这种事也是难免。地藏像会原谅人们的吧。

: 

设计上倒是很直接地设计成笠地藏了啊。

: 

关卡本身就是冬天啊。应该是有人给她盖上的吧。
——
: 

「永夜抄」里魔理沙倒是有这么一句台词 (未找到链接)「还有跑了一圈给很多地藏都打了伞，全部都是爱丽丝干的」来着。

竟然还有这么久远的伏笔啊（笑）。成美已经不再是地藏像，从存在上来说已经是魔法使了。她住在魔法森林，接受了魔力的影响变成令人难以搞明白的存在了。已经没太多地藏的感觉了。
