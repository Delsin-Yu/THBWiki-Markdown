东方绯想天 | 原作游戏解锁方式

-  **剧情模式** (Story Mode)有15个可用角色（一开始只有3个可用角色，有12个角色需解锁）
-  **对战模式** (Adcade Mode/Vs Com/Vs Player/Vs Network)有15个可用角色（一开始只有13个可用角色，有2个角色需解锁）  


  
各锁定角色解锁方式:  

  

博丽灵梦
:  **剧情模式** ：初期可使用  
 **对战模式** ：初期可使用  

雾雨魔理沙
:  **剧情模式** ：初期可使用  
 **对战模式** ：初期可使用  

十六夜咲夜
:  **剧情模式** ：初期可使用  
 **对战模式** ：初期可使用  

爱丽丝·玛格特洛依德
:  **剧情模式** ：完成雾雨魔理沙的剧情模式后解锁  
 **对战模式** ：初期可使用  

帕秋莉·诺蕾姬
:  **剧情模式** ：完成十六夜咲夜的剧情模式后解锁  
 **对战模式** ：初期可使用  

魂魄妖梦
:  **剧情模式** ：完成博丽灵梦的剧情模式后解锁  
 **对战模式** ：初期可使用  

蕾米莉亚·斯卡蕾特
:  **剧情模式** ：完成博丽灵梦、雾雨魔理沙、十六夜咲夜的剧情模式后解锁  
 **对战模式** ：初期可使用  

西行寺幽幽子
:  **剧情模式** ：完成蕾米莉亚·斯卡蕾特的剧情模式后解锁  
 **对战模式** ：初期可使用  

八云紫
:  **剧情模式** ：完成至少四个角色的剧情模式后解锁  
 **对战模式** ：初期可使用  

伊吹萃香
:  **剧情模式** ：完成至少七个角色的剧情模式后解锁  
 **对战模式** ：初期可使用  

铃仙·优昙华院·因幡
:  **剧情模式** ：完成爱丽丝·玛格特洛依德的剧情模式后解锁  
 **对战模式** ：初期可使用  

射命丸文
:  **剧情模式** ：完成蕾米莉亚·斯卡蕾特的剧情模式后解锁  
 **对战模式** ：初期可使用  

小野塚小町
:  **剧情模式** ：完成魂魄妖梦的剧情模式后解锁  
 **对战模式** ：初期可使用  

永江衣玖
:  **剧情模式** ：  
 **对战模式** ：完成博丽灵梦、雾雨魔理沙、十六夜咲夜中任意一个的剧情模式后解锁  

比那名居天子
:  **剧情模式** ：  
 **对战模式** ：完成博丽灵梦、雾雨魔理沙、十六夜咲夜中任意一个的剧情模式后解锁
