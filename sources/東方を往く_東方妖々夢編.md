## 评论
- 作者

```
全国各地の東方っぽいところを訪ね歩く本第五弾。10周年を迎えた東方妖々夢の舞台を巡ります。
```

- 蜜瓜

```
全国各地の東方にゆかりのある場所を深く掘り下げる1冊、今回は妖々夢の舞台を巡ります!
道中BGMに関連した遠野物語の逸話から、周辺の地域に伝わる河童のかわいいエピソード、
遠野市の名産の他に観光解説や交通ガイドなど、読み応えのある濃密なコンテンツは必見☆
聖地巡礼や旅行に欠かせない、ファンならずとも必携のガイドブックをたっぷりお楽しみ下さい。
```
