## 技能加点
  
姆Q常见加点流派 （由“3倍I★SCREAM”提供）
  


### 主金流
```
 以金元素为主要加点的姆Q流派。多用于团队缺少廉价输出时加点使用，以水银剧毒（水金）和陨石（金土）为中心的一套爆发流加点,其次生命收割者(金木)也有着不俗的伤害 而且CD短，可以用来在残血的适合进行K头或击杀炮车或其他敌方生物；金属疲劳的伤害在前期也有不错的收益（但除非是机动性极低并且没用控制或保命的机体，个人建议最好不要拿来先手，在第一套日符打出去之后来根据敌方血量进行残血收割）  
  

```

  
对线期加点方面有两种选择:

 三级时则为姆Q的一个质变时期，因为这时候你将可以进行日符「皇家烈焰」的释放,此视根据你二级进行的加点有木就加水，有水就加木，此时你会获得一个额外的技能点数，这个点数依旧是根据情况来进行加点。你可以选择将这个点加到水或者是土:<br><br>
 如果选择加水，那么将意味着你线上则会有更多的回蓝，可以进行多次的消耗与回复，甚至可以不需要依靠御币进行蓝量回复；同时你的湖葬将延长持续时间，可以降低你搓日符时的容错率；<br><br>
 如果你选择加土，那么则意味着你线上牺牲了额外的回蓝而进行控制与伤害的加深，此时你的水精灵将无法回复更多的蓝量，也就是说你的水精灵的效果此时是在大约不影响蓝量的情况下额外释放一个回血技能，即释放一个没什么用的技能来达成日符的效果——但是你的缠绕控制时间与三石塔的控制时间与伤害都会进行增加，从而使你三级造成的伤害配合日符更加提高对线时的斩杀线。<br><br>
 装备上建议随身携带御币或丰收果篮加蘑菇或者蓝瓶，衰人卡也不能断，在拉高斩杀线的同时还能进行蓝量回复。<br><br>
 四级时的加点则比较随意根据情况来判断，3水1木1土，2水2木1土或2水1木1土都可以 (个人一般比较喜欢2水1木2土的加点，有蓝有输出，前期1秒的角笛时间和1级木的回血也勉强够，如果不够就先出御币保证续航)。<br><br>

### 以火转金流
```
 以火转金流：也就是所谓的火系帕秋莉≈火卡（和火卡不同的是，你点火没有天火）多常用于中单姆Q的加点。不同于前面的主金流，火卡一般出山时间为20min多左右 也就是说以多刷为主，如果无法保证前期队友野区打架能拿到头，建议不要去帮忙，以稳定自己节奏为中心，20min后让队友以你为中心带领团队打架。  
  

```

<pre> 对线加点方面：出门不用考虑，确定自己打中的话一定要学火! 一定要学火! 一定要学火!  对线召唤火人双操补刀与压制。（如果你不学火 我觉得你可能大部分时候连刀都补不了）<br><br>
 二级的时候就可以学水，这时你可以选择继续火人补刀并且时不时给对面上燃素之雨以此来消耗对面血线，并且在不补刀的时候切换至水来加一些回蓝，当补刀时再切换回火增加攻击力以进行补刀。<br><br>
 三级的时候学习一级木元素，并且用额外点数再点一级火。此时你手上将会有火人 燃素 森林大火 水精灵 湖葬 风铃的角笛，这时中单的话3级也就大概两三分钟左右。如果说对面有gank型的酱油（如：天子、古明地觉、早苗之类的游走型辅助)此时你可以选择有线推线，没线打野的状态。2级的火算准兵线到中路的时机直接释放，放完就去请小野或对面野区离中路近的中野（这边比较建议用森林大火[火木]来快速清理线上小兵，因为森林大火相比燃素之雨，伤害高，持续时间长，范围固定，释放完即可转头去打野，而燃素之雨可能在持续时间内无法杀死小兵，所以可能会吃不到小兵补刀经济，建议拿来刷野的时候用)。<br><br>
 四级的时候学两级水，保证清野然后回蓝，这样可以不用过多经济来出回蓝装备，尽早刷出钱箱然后做礼服。<br><br>
 五级的时候点三级火，此时后面就可以不需要再点更多的火了，三级火足以保证烧死兵线和火人清野的速度，保证自己的快速发育。<br><br>
 六级的时候点一级金和一级土，这时你的金木水火土都有了，但此时依旧是以刷为主以掩护队友为辅，尽量不要去对面找事，多多发育，争取10min的时候能把钱箱礼物做出来，13min的时候合成礼服。<br><br>
 慢慢的刷，大概20min多的时候，你金等级差不多刷满了，水有大概三四级，木也有两三级，三级火，一级土，做出你的第一个大件——羊刀。此时则是你出山的时候。并且20min后一定是白天，你将迎来你的强势期。配合队友进行打架，，如果队友不打，则继续用火刷野推兵推塔，或者尝试自己去单抓对面少女保证一个中期节奏压制，此时你的装备大致为 钱箱 礼服 羊刀 鞋子 再加个鞋垫或御币之类的东西。并且卡可以换成逗人卡，此时你的伤害已经不需要衰人卡来弥补了，所以留一手保命卡就行。<br><br>
 后续加点水可以保持四级不变或五级，然后副点木系保证生存和角笛持续时间以方便进行gank，土可以看情况点两级，火不变。<br><br>
</pre>
</td></tr></tbody></table>
