# 东方兽王园/角色解锁

东方兽王园 | 原作游戏解锁方式

  
本作拥有19个可用角色，但一部分角色需要完成特定条件才能解锁，且不同模式下的解锁条件不同。
  

## 剧情模式（Story Mode）解锁条件
博丽灵梦
: 初始可用。
雾雨魔理沙、东风谷早苗
: 完成博丽灵梦的剧情后解锁。
八云蓝
: 在剧情线中遭遇并击败以解锁。  
可用剧情线：雾雨魔理沙、东风谷早苗
高丽野阿吽
: 在剧情线中遭遇并击败以解锁。  
可用剧情线：东风谷早苗
娜兹玲
: 在剧情线中遭遇并击败以解锁。  
可用剧情线：雾雨魔理沙
清兰
: 在剧情线中遭遇并击败以解锁。  
可用剧情线：雾雨魔理沙
火焰猫燐
: 在剧情线中遭遇并击败以解锁。  
可用剧情线：八云蓝
菅牧典
: 在剧情线中遭遇并击败以解锁。  
可用剧情线：八云蓝、清兰
二岩猯藏
: 在剧情线中遭遇并击败以解锁。  
可用剧情线：东风谷早苗
吉吊八千慧、骊驹早鬼、饕餮尤魔
: 完成八云蓝、高丽野阿吽、娜兹玲、清兰、火焰猫燐、菅牧典、二岩猯藏剧情后解锁。
伊吹萃香
: 完成豫母都日狭美剧情后解锁。
孙美天、三头慧之子、天火人血枪
: 完成吉吊八千慧、骊驹早鬼、饕餮尤魔剧情后解锁。
豫母都日狭美
: 完成孙美天、三头慧之子、天火人血枪剧情后解锁。
日白残无
: 完成伊吹萃香剧情后解锁。

## 对战模式（VS Mode）角色解锁条件
博丽灵梦、雾雨魔理沙、东风谷早苗、八云蓝、高丽野阿吽、娜兹玲、清兰、火焰猫燐、菅牧典、二岩猯藏
: 初始可用。
吉吊八千慧、骊驹早鬼、饕餮尤魔、伊吹萃香、孙美天、三头慧之子、天火人血枪、豫母都日狭美、日白残无
: 使用任意角色在剧情模式中遭遇并击败对应角色后解锁。

### 在线对战模式（Online VS Mode）
  
完成博丽灵梦、雾雨魔理沙、东风谷早苗、八云蓝的剧情后解锁。
