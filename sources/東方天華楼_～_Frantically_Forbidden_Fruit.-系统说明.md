## 基本操作
  
方向键：控制自机移动
  
  
Z键：射击/确定，长按跳过对话
  
  
X键：bomb/取消，天华统一模式没有B的情况下饮用果汁
  
  
C键：天华统一模式下饮用果汁
  
  
Shift：低速
  
  
Esc：暂停
  
  
Ctrl：长按跳过对话
  
  
暂停界面按R：重开
  
  
P：截图
  
  
Alt+Enter：窗口/全屏模式切换
  

- 通过config.exe可以改键。


## 道具说明

### 火力道具
小红点
: 增加0.01p。
中红点
: 增加0.05p。
大红点
: 增加1.00p。在自动收点时记作100个道具。
F点
: 恢复所有Power。在自动收点时记作500个道具。

- 残机减少时会扣除0.50p并掉落5个小红点和3个中红点（共0.20p），当剩余残机为0时会额外掉落F点。大红点只能通过敌人掉落获得。
- 当Power不为满（5.00）时，高速状态下击落敌人会掉落额外的小红点。


### 分数道具
蓝点
: 分数道具。在自动收取的情况下获得等同于当前最大得点值的分数，非自动收取的情况下依照屏幕高度结算得分。
绿点
: 消弹/击打道具，增加最大得点。在自机低速射击或者满Power射击时会出现。


### 果汁槽道具
擦弹道具
: 待在子弹旁边一段时间后会出现（参见绀珠传），增加5graze。颜色分别为红橙黄绿青蓝紫白八种，拾取会填充相应颜色的果汁槽，并且增加最大得点。
水果道具
: 通过收点结算、击破敌人或者章节结算时出现，分为苹果、橘子、香蕉、西瓜、蜜瓜、葡萄、桃子、杨桃八种。每种水果会增加不同值的残机/符卡槽，并且填充相应颜色的果汁槽。


### 残机/符卡道具
残机道具
: 获得一个残机，并恢复全部生命值。
残机碎片
: 恢复最大生命值的一半。
符卡道具
: 获得一个符卡。
符卡碎片
: 恢复等同于低费用的符卡消耗值的符卡槽。


## 游戏模式
  
游戏分为两个模式，分别是天华统一模式和传奇模式。无论哪个模式下残机和bomb的上限个数都是5。
  
  
天华统一模式下，残机将以hp条的方式计算，通过奖残扩增生命最大值，奖残时恢复所有体力。不同的子弹伤害也不一样，power会减少相应的伤害值。
  
  
当hp小于0时，残机减一，hp回复至满。进入下一关或者通过果汁的效果也可以增加hp。
  
  
除去残机道具之外，还可以通过打分奖励获取残机。分别在1.5亿，3.5亿，7.5亿，11亿，22亿，33亿，44亿，55亿，66亿，77亿，99亿9999万各获得一个残机。
  
  
传奇模式下，果汁系统被禁用，残机按照常规的残机系统进行计算。通过收集水果道具可以增加残机槽，当残机槽满的时候会奖励一残。不同的水果道具增加的残机槽也不同。
  


## 道具回收奖励
  
处于回收线上时，将主动回收所有道具。此时，自机上方会出现数字和回收条。丢B会同样会自动回收道具。
  
  
当回收的道具个数满足分母值，且回收条变为0的时候，将会出现分数奖励，并且放出水果道具。
  
  
水果道具是按顺序出现的。
  


## 异常状态
  
有一部分的子弹击中自机后，会引起异常状态。状态图标会显示在自机附近并持续一段时间。
  
  
饮用果汁的时候，会自动解除异常状态。
  
  
传奇模式下不仅会对自机造成伤害，而且还会减少残机槽。
  


### 状态一览
毒
: 移动速度下降，持续受到伤害。
烧伤
: 射击威力下降，持续受到伤害。
麻痹
: 移动速度下降，射击频率降低。
冰冻
: 持续受到少量伤害，移动速度上升。
混乱
: 移动速度变为随机。
猛毒
: 移动速度变得非常慢，受到大量伤害。
睡眠
: 由于自机睡着了所以不能移动。
恢复
: 一定时间内持续恢复体力。（只在桃子果汁饮用后才会出现这个状态。饮用新的果汁不会打断恢复状态。）


## 果汁系统
  
仅在天华统一模式下才能使用。
  
  
画面左侧有一个进度条，每条进度条最多能容纳2.0L的果汁。当进度条满的时候，该果汁的颜色被确定为最后一个回收的道具颜色，并且存入左下角的空瓶子当中；同时进度条清空进行下一条果汁的储存。最多可以存储3瓶果汁（6.0L）。
  
  
当持有1.0升及以上果汁的时候，可以按C键发动果汁的特效。发动时间内自机无敌，并且会使符卡收取分数大幅度上涨。多种果汁的效果可以同时发动，持续时间与该种类果汁的量正相关。
  
  
部分果汁在持续时间内受击会减少持续时间。触发条件目前不明。
  
  
果汁的味道由进度条下方的星状图决定。星状图较多的一项为当前果汁的味道，可以通过收取其他水果道具或者擦弹来改变星状图，从而改变果汁味道。
