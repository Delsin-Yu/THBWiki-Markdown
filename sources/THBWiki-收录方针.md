## 重要事项
- 本站是东方Project主题维基百科，除 **特别允许的作品** ，基于其他作品创作或纯原创的作品一般不在收录范围内。
- 请确保编写内容属于公有领域、合理使用、或处于与署名-非商业性使用-相同方式共享 3.0协议相容的许可下。
- 本站不是作品发布平台，请先在其他平台或实体方式发布作品，然后再在本站记录作品相关信息。
- 请尊重他人隐私。
- 如无作品原作者的允许，请勿添加作品本体的下载链接。原作者发布的无门槛下载链接视为原作者已允许，但也请尽量放原链接，不要通过第三方平台转存。下载或获取下载链接时需要用户登录、加群、付费或非公开密码等操作时视为有门槛。
- 如对本方针有疑问或建议，可以在讨论板 (未找到链接)提出。

## 用语
东方官方
: [ZUN](./ZUN.md)及[上海爱丽丝幻乐团](./上海爱丽丝幻乐团.md)。
东方
: 泛指由 **东方官方** 参与创作的[东方Project系列作品、音乐CD、西方Project系列作品、及其他ZUN参与的作品](./官方作品.md)。
东方相关元素
: 存在于 **东方** 里的独特内容及元素，例如角色、台词、设定、世界观、道具、音乐、游戏内容等。元捏他、二次衍生捏他不在此范围内。
作品内容
: 文字内容、图画内容、音乐内容、视频内容。
作品的任意部分
: 封面、名称、 **作品内容** 、作品设定、作品概念。作者、制作社团、贩卖途径不在此范围内。
东方同人作品
: 在 **作品的任意部分** 包含任意 **东方相关元素** 或基于任意 **东方相关元素** 二次创作作品。
东方主题活动
: 以 **东方** 为唯一主题的活动。
东方圈
: 围绕 **东方** 产生的社群、创作者群体、及ZUN相关的人物。
个人隐私
: 真实姓名、肖像、性别、住址、联络方式等跟个人身份和私密活动相关的信息。
特别允许的作品
: 只有[连缘Project](./连缘Project.md)。特别允许的作品在以下收录范围方针下享有和 **东方** 同样的待遇。

## 收录范围
### 东方官方作品
- 由 **东方官方** 创作的内容原则上无收录限制。

#### 角色二次设定
- 二次设定必须能够写出可信且易于理解的发源时间、流行范围、起始原因（如果适用）。
- 二次设定的流行范围必须是公开的公众的，而非仅限个人或私密的小群体。
- 优先收录全年龄、易于大众接受的二次设定。

#### 捏他
- 捏他必须与 **东方相关元素** 相关。
- 不建议写借用自非 **东方** 作品里的捏他。

### 东方同人作品
- 作品必须符合 **东方同人作品** 定义。
- 作品必须已经实际发布，或已确定将会在指定日期或活动中实际发布正式版。
- 作品不能仅使用原作素材。
- 作品不能包含大量抄袭内容。
- 如果作品有不同版本的区分（例如初回限定版、平装版、特装版等），仅按照最初发布的版本的内容收录。如版本之间的 **作品内容** 有实质区别，则可以分开收录不同版本。
- 如果作品包含作者设计的隐藏内容，视情况略过收录隐藏的内容。
- 如果作品实际内容和作者宣称的不一样，按照实际情况收录。
- 原则上收录任何限制级的作品，但必须使用模板参数明确标记。

#### 独立收录作品
- 作品属于以下类型并符合该类型的收录条件则允许以独立词条形式收录：

: - 同人志 (未找到链接)（漫画、画集、小说）
- 专辑 (未找到链接)（曲目）
- 软件 (未找到链接)（游戏）
- 周边 (未找到链接)（手办）
- 视频 (未找到链接)（动画）


- 不符合独立收录要求的作品按照[非独立收录作品](#非独立收录作品)规则处理。

##### 同人志
- 由于制作门槛较低，本站要求作品必须已以实体印刷方式发布。如果仅在网络上发布，则需要有足够知名度或影响力才予以收录（仅收录基本信息，可外链至阅读地址），请勿将全文收录在Wiki中，以便防止产生不必要的麻烦。
- 仿游戏风格编写的设定集类作品若符合以下条件，可按照同人游戏格式编写，但仍然需使用同人志信息模板并标记类型为设定集：

: - 作品包含设定文档、关卡对话、自机角色设计和敌对角色设计。
- 作品已在其他平台或以实体方式发布。
- 作品文本有较高完成度。


##### 专辑
- 作品必须包含至少一首使用 **东方** 原曲的同人曲目，若不符合则视为原创专辑，按照原创专辑规则处理。
- 原创专辑及原创曲目若符合以下任一条件则予以收录：

: - 作品资料太少未能判断原曲信息，但实际上应该是 **东方同人作品** 。
- 作品是仅为另一符合收录条件的作品而制作的附属作品，例如原声集、专属背景音乐等。
- 作品是根据任意 **东方相关元素** 的印象创作的。
- 作品是围绕任意 **东方相关元素** 的主题创作的。
- 作品是仿东方风格的原创曲目。
- 作品是 **东方相关元素** 主题的无背景音乐配音作品。


- 如果专辑内有其他非东方原曲或原创曲目，那些曲目在专辑词条内正常收录，但不会创建相关原曲及歌词词条。
- 以专辑形式实体发布和网络发布的作品以同样规则处理。
- 若作品仅在网络上以独立曲目形式发布，则仅允许以同作者为基础创建曲目合集词条（按照同人专辑格式），并仅收录使用 **东方** 原曲的同人曲目。

##### 软件
- 作品必须以可以独立运行或进行游玩，若必须依附于其他软件或游戏，则视为MOD，按照MOD规则处理。
- MOD若符合以下任一条件，且不违反原软件的使用协议，则予以收录：

: - 对原软件改动较大，实质上已经转化成一个独立软件。
- 在原游戏基础上添加了大量 **东方相关元素** 内容。


- 游戏体验版可以和正式版分开收录。
- 实体发布和网络发布的作品以同样规则处理。
- 仿游戏风格编写的设定集类作品按照同人志规则处理。
- 聊天机器人与基于人工智能调教的聊天人格均不属于收录范围内。

##### 周边
- 作品必须已做成实体产品发布。

##### 视频
- 由于制作门槛较低，本站要求作品必须已以实体印刷方式发布。如果仅在网络上发布，则需要有足够知名度或影响力才予以收录。

#### 非独立收录作品
- 若移除完成度、知名度及实体发布相关的要求后，作品符合其作品类型的收录条件，则允许以文字描述方式收录。
- 仅限简单文本介绍，禁止摘录作品内容，禁止上传作品相关图片。

### 二次创作角色
- 角色必须在符合[东方同人作品独立收录要求](#独立收录作品)的作品中作为登场角色出现，并拥有角色自己的设定或对话内容。若不符则视为原创角色，按照原创角色规则处理。
- 原创角色若符合以下任一条件则予以收录：

: - 角色是任意 **东方** 原作发售前创作的钓鱼角色，并拥有角色自己的设定或对话内容。
- 角色是 **东方圈** 知名人士的自设角色，并拥有角色自己的设定或对话内容。


- 不符合收录规范的原创及自设角色，可以写在个人的用户Wiki页里。

### 同人社团
- 同人社团必须符合以下任一条件：

: - 同人社团是任何符合[东方同人作品独立收录要求](#独立收录作品)的作品的制作方。
- 同人社团是任何符合[活动收录要求](#活动)的活动的举办方。
- 同人社团是 **东方官方作品** 的制作方。
- 同人社团与 **东方官方** 有直接关联。


- 可以在同人社团词条中描述[非独立收录作品](#非独立收录作品)。

### 商业公司
- 商业公司的收录方针与同人社团相同。

### 人物
- 人物必须符合以下任一条件：

: - 人物必须是任何符合[东方同人作品独立收录要求](#独立收录作品)的作品的制作人员。
- 人物是符合[同人社团收录要求](#同人社团收录范围)的同人社团的主要成员。
- 人物是 **东方圈** 知名人士。
- 人物是经常公开翻唱或游玩 **东方同人作品** 的主播或虚拟主播。


- 请勿在人物词条中披露未经当事人允许公开的 **个人隐私** 。若当事人已主动在本站以外的公共网络空间披露该信息，则视为已允许公开。
- 请勿在人物词条中编写与 **东方** 无关的经历、人际关系和个人成就。
- 可以在人物词条中描述[非独立收录作品](#非独立收录作品)。

### 活动
- 活动必须是 **与东方Project有关** 的线下 **公开** 活动。
- 对于 **有关** 的定义如下，活动必须符合以下条件中的任意一个：

: - 活动是 **东方主题活动** 。
- 活动为多主题，但在主办方对外宣发中 **以东方Project作为首要主题** 的线下活动。
- 不符合上述条件，但有符合[独立收录作品](#独立收录作品)以该活动的某一届进行首次发布。
- 不符合上述条件，但是 **博丽神主ZUN** 作为该活动的嘉宾或出演参与了该活动。


- 对于 **公开** 的定义如下，活动必须符合以下条件中的任意一个：

: - 活动 **公开** 对外售票。
- 活动在 **可链接的公共网络** 上公开地址，并开放入场（需要面向所有东方爱好者群体，仅可以有着装、性别等作品IP或公序良俗方面的限制）。
- 活动是舞台表演、讲座、访谈性质，并在公共网络上进行直播、录播或文字抄录。


- 活动必须满足以下条件中的一个或以上：

: - 该活动已经 **成功举办** 过至少一届（需要有 **网络评价** 信源或者 **现场照片** 信源证实）。
- 该活动尚未举办，但已经进行 **公开售票** （需要有 **票务网站** 信源证实）。
- 该活动尚未举办，但已经进行 **公开宣传** （需要有 **可链接的SNS平台** 信源证实）。


- 符合以下任意一个情况的活动，将不予收录，如果已建立词条，管理员有权删除或归档该词条：

: - 该活动从未成功举办过，且并未公开确定的举办地点与举办时间。
- 该活动从未成功举办过，且已经宣布无限期延期。
- 该活动从未成功举办过，且该活动已经超过3个月没有任何公开宣发。
- 该活动从未成功举办过，且该活动宣称的首届举办时间距今超过12个月。
- 通过公开平台或可查证的信源，可以确认该活动主办方并无实际举办意向。
- 活动的官方正式名为【××(地名)的东方主题活动】，与THBWiki保留语法冲突。


### 事件
- 事件必须符合以下任一条件：

: - 事件涉及ZUN参与。
- 事件对 **东方圈** 产生重要影响。


- 请勿在事件词条中披露未经当事人允许公开的 **个人隐私** 。若当事人已主动在本站以外的公共网络空间披露该信息，则视为已允许公开。

### 商业游戏
- 游戏中必须包含由游戏制作方提供的基于 **东方相关元素** 创作的可游玩内容。
