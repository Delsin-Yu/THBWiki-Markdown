```
東方プロジェクトのアレンジアルバムです。
伴天連のドグラマグラ、幻魔怪奇の世界、
かぐや姫と化けウサギ達の、日本の祟りの音楽を表現しています。
今回は打って変わって、和風・中国王朝風のアレンジに。歌も収録予定です。
-----
原曲のみならず、原作・東方永夜抄に忠実 ( 私解釈では。 ) に製作しているアルバムで、
日本のジメジメとした陰陽・幽玄・祟りのイメージの音楽です。
多分このアルバムは、この辺りでは最初で最後の珍しい類のものになりそうだと思っています。
```
